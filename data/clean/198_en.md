# Images Resizing

The maximum allowed size for images to be uploaded to the Expat Management System is 200 KB.

In order to resize your images to the specified size, you can use the programme [Paint.net](https://www.getpaint.net/download.html), which can be downloaded online.

To resize the image, please follow these steps:

1. Click on the image then on resize.
2. Specify the height and width as shown below.
3. Click on File, then Save as or Ctrl+Shift+S and save your image as JPEG format. The image is ready to be uploaded with the application.

Please make sure that the document size is not more than 200 kb

(You can check the document size after saving the document by right-clicking on the document and then clicking on Properties, same as shown below).

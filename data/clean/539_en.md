# Application for a New Work Permit for Golden Residency Holders

Through this service, Golden Residency Holders who wish to work, can obtain the necessary work permit.

## Important Notes

- The work permit duration for Golden Residency Holders is for one or two years.
- Employers will be able to renew the work permit 6 months prior to its expiry.
- If the Golden Residency Holder already has an existing Work Permit prior to acquiring a Golden Residency, then once the work permit is due for renewal, the employer must cancel the existing work permit and apply for the Golden Residency Holder work permit (Cancellation to be applied via eSupport function via the [Expat Management System (EMS)](https://www.lmra.gov.bh/EMS_Web/)).
- If the Golden Residency Holder is an investor and is one of the partners of the establishment, he/she does not have to apply for a work permit for Golden Residency Holders.
- If there are no available work permits under the Commercial Registration (CR) according to the Bahrainization percentage, an additional fee for the parallel system will be applied.

## Process Description

### Physical Location

1. Book an appointment via [Appointment Management System (AMS).](https://ams.lmra.gov.bh/)
2. Download [Permit/ Service Application Form PDF (1.39 MB)](https://lmra.gov.bh/files/cms/shared/permit-service-application-form%20(1).pdf) and fill in the required information.
3. Submit the application form along with the required documents at the Client's Services counters.
4. Once the application is submitted, LMRA employee will inform the client that he/ she will need to pay the the admin fees (Either on counter by using the Point of Sale (POS) machine, or via his E-Key on [Expat Management System (EMS)](https://www.lmra.gov.bh/EMS_Web/).
5. Upon completion of the admin fees payment, the application will be transferred for verification.
6. You can check the application status (whether it has been approved or rejected) by:
  - Calling LMRA's call centre on [+973 17506055](tel:+97317506055).

7. If the application is approved, the expatriate employee needs to undergo the [biometric data collection](https://www.lmra.gov.bh/en/page/show/187) and then need to pay the work permit fees via the [Expat Management System (EMS)](https://www.lmra.gov.bh/EMS_Web/) or by visiting one of the [accredited banks](https://www.lmra.gov.bh/en/page/show/212).

### eService (eSupport function) via the [Expat Management System (EMS)](https://www.lmra.gov.bh/EMS_Web/)

1. Submit your request via the eSupport function on the [Expat Management System (EMS)](https://www.lmra.gov.bh/EMS_Web/).
2. By attaching the required forms and documents in 1 pdf file (Does not exceed 3 MB).
  - If the expatriate employee has a valid work permit, please attach the [Termination Declaration Form PDF (1 MB)](https://www.lmra.gov.bh/files/cms/downloads/english_attachment/termination_declaration_form.pdf) so that the permit is cancelled by LMRA, and you will not be charged for the admin fee of the cancellation.
  - Note: In case the expatriate employee has dependants, you will need to include them in the termination declaration form and attach a copy of their Golden/ Platinum visas.

3. To apply for the work permit for Golden/ Platinum visa holders, fill in the [Permit/ Service Application Form PDF (1.39 MB)](https://lmra.gov.bh/files/cms/shared/permit-service-application-form%20(1).pdf) signed by the authorized person.
4. Once the application is applied, you will be informed via the e-ticket that it is awaiting admin fees payment.
5. Admin fees to be paid via the [Expat Management System (EMS)](https://www.lmra.gov.bh/EMS_Web/).
6. Upon completion of the admin fees payment, the application will be transferred for verification.
7. You can check the application status (whether it has been approved or rejected) by:
  - Calling LMRA's call centre on [+973 17506055](tel:+97317506055).

8. If the application is approved, the expatriate employee needs to undergo the [biometric data collection](https://www.lmra.gov.bh/en/page/show/187) and then you are required to pay the work permit fees via the [Expat Management System (EMS)](https://www.lmra.gov.bh/EMS_Web/) or by visiting one of the [accredited banks](https://www.lmra.gov.bh/en/page/show/212).

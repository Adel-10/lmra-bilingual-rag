# Renewal of the Work Permit for Golden Residency Holders

Through this service, employers will be able to renew the work permit for Golden Residency Holders 6 months prior to its expiry. It can be renewed for 1 year or 2 years.

## Required Documents

**Renewal of the Work Permit for Golden Residency Holders Required Documents**

| No. | Document | Document Type |
|---|---|---|
| 1 | Permit/ Service Application Form PDF (1.39 MB) | Copy Expatriate employee declaration section in the form must be signed by the employee. |

## Process Description

### Physical Location

1. Book an appointment via [Appointment Management System (AMS).](https://ams.lmra.gov.bh/)
2. Download [Permit/ Service Application Form PDF (1.39 MB)](https://lmra.gov.bh/files/cms/shared/permit-service-application-form%20(1).pdf) and fill in the required information.
  - Note: Expatriate employee declaration section in the form must be signed by the employee.

3. Submit the application form along with the required documents at the Client's Services counters.
4. Once the application is submitted, LMRA employee will inform the client that he/ she will need to pay the the admin fees (Either on counter by using the Point of Sale (POS) machine, or via his E-Key on [Expat Management System (EMS)](https://www.lmra.gov.bh/EMS_Web/).
5. Upon completion of the admin fees payment, the application will be transferred for verification.
6. You can check the application status (whether it has been approved or rejected) by:
  - Calling LMRA's call centre on [+973 17506055](tel:+97317506055).

7. If the application is approved, you need to pay the work permit fees via the [Expat Management System (EMS)](https://www.lmra.gov.bh/EMS_Web/) or by visiting one of the [accredited banks](https://www.lmra.gov.bh/en/page/show/212).

### eService (eSupport function) via the [Expat Management System (EMS)](https://www.lmra.gov.bh/EMS_Web/)

1. Download and fill out the [Permit/ Service Application Form PDF (1.39 MB)](https://lmra.gov.bh/files/cms/shared/permit-service-application-form%20(1).pdf) and submit your request via the eSupport function on the [Expat Management System (EMS)](https://www.lmra.gov.bh/EMS_Web/) by attaching the required form in JPEG format (Does not exceed 150 KB).
  - Note: Expatriate employee declaration section in the form must be signed by the employee.

2. To apply for the work permit for Golden/ Platinum visa holders, fill in the [Permit/ Service Application Form PDF (1.39 MB)](https://www.lmra.gov.bh/files/cms/downloads/english_attachment/permit-service-application-form.pdf) signed by the authorized person.
3. Once the application is applied, you will be informed via the e-ticket that it is awaiting admin fees payment.
4. Upon completion of the admin fees payment, the application will be transferred for verification.
5. You can check the application status (whether it has been approved or rejected) by:
  - Calling LMRA's call centre on [+973 17506055](tel:+97317506055).

6. If the application is approved, you need to pay the work permit fees via the [Expat Management System (EMS)](https://www.lmra.gov.bh/EMS_Web/) or by visiting one of the [accredited banks](https://www.lmra.gov.bh/en/page/show/212).
7. Upon completion of the admin fees payment, and If you would like to receive the renewal certificate, kindly open a new ticket via the eSupport function in the [Expat Management System (EMS).](https://www.lmra.gov.bh/EMS_Web/)

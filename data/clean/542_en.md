# Change of Occupation for Golden Residency Holders

Through this service, employers can apply to change the occupation of the Golden Residency Holders.

## Service Conditions

- For occupations that require specialization (medical occupations, educational occupations, and engineers), the applicant must submit the approval from the concerned entity.
- The work permit must be valid.

## Required Documents

**Change of Occupation for Golden Residency Holders Required Documents**

| No. | Document | Document Type |
|---|---|---|
| 1 | Employment contract is optional (Provided that it contains Arabic language) | Original |
| 2 | Approval from the concerned entity for specialized occupations (Medical occupations, educational occupations and engineers) | Original |

## Process Description

### Physical Location

1. Book an appointment via [Appointment Management System (AMS).](https://ams.lmra.gov.bh/)
2. Download [Permit/ Service Application Form PDF (1.39 MB)](https://lmra.gov.bh/files/cms/shared/permit-service-application-form%20(1).pdf) and fill in the required information.
3. Submit the application form at the Client's Services counters.
4. Once the application is submitted, it will be transferred for verification.
5. You can check the application status (whether it has been approved or rejected) by:
  - Calling LMRA's call centre on [+973 17506055](tel:+97317506055).

6. If the application is approved, the change of occupation fees will be added to the establishment’s monthly invoice which can be paid via the [Expat Management System (EMS)](https://www.lmra.gov.bh/EMS_Web/) or by visiting one of the [accredited banks](https://www.lmra.gov.bh/en/page/show/212).

### eService (eSupport function) via the [Expat Management System (EMS)](https://www.lmra.gov.bh/EMS_Web/)

1. Download and fill out the [Permit/ Service Application Form PDF (1.39 MB)](https://lmra.gov.bh/files/cms/shared/permit-service-application-form%20(1).pdf) and submit your request via the eSupport function on the [Expat Management System (EMS)](https://www.lmra.gov.bh/EMS_Web/) by attaching the required form in JPEG format (Does not exceed 150 KB).
2. Once the application is applied, you will be informed via the e-ticket that it has been transferred for verification.
3. You can check the application status (whether it has been approved or rejected) by:
  - Calling LMRA's call centre on [+973 17506055](tel:+97317506055).

4. If the application is approved, the change of occupation fees will be added to the establishment’s monthly invoice which can be paid via the [Expat Management System (EMS)](https://www.lmra.gov.bh/EMS_Web/) or by visiting one of the [accredited banks](https://www.lmra.gov.bh/en/page/show/212).

## Fees

- Service Fee: BHD 10 (Change of occupation fee will be added to the establishment’s monthly invoice).

## Processing Time

- 3 working days.
